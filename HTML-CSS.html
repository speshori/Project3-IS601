How the Internet Works



W3C

The W3C or the World Wide Web Consortium is the organization that sets the standards for the World Wide Web (WWW). It was actually founded in 1994 by Tim Berners-Lee and is currently still led by him. It was founded at MIT with support from the European Commission and DARPA (Defense Advanced Research Projects Agency). They set standards over the Internet Protocols for many different languages inclusive of HTML, CSS, DOM, JSON-LD, XML, and XHTML.

HTML

HTML stands for Hypertext Markup Language and in it of itself is not a programming language as it does not have the ability to creat dynamic functionality, but instead is used to organize a page for example or website like a Word Document. HTML was invented by Tim Berner's Lee. In the H for HTML which means Hypertext, it stands for text that contains references (links) to other texts that viewers can access immediately. HTML is also considered the official web standard. It is maintained by the W3C as well which was also founded by Tim Berners-Lee.

CSS

CSS stands for Cascading Style Sheets and is used to interface with HTML as HTML builds the structure of your page, CSS will detail it and its layouts. CSS is able to interact with a given HTML element and apply certain attributes and appearances to it. CSS can apply to one element or many, it can also be written inline on an HTML page or be stored separately and imported into an HTML page to make those changes on that given page.
