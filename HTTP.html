How the Internet Works



HTTP and HTTPS

HTTP or HyperText Transfer Protocol is a protocol in which hypertext is transferred over the internet. It is a simple and widely used protocol, but the issue lies within this simplicity as HTTP is simply sent over the web as plain text. Hence why HTTPS was introduced. HTTPS sets up a secure connection between the server and browser. It takes on a protocol such as SSL or TLS and is able to utilize a private on behalf of the web server to maintain security. HTTP also utilizes the well known port 80 while HTTPS uses 443.
