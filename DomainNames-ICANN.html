How the Internet Works



Domain Names

At its core a domain name is the name of a website. It is also referred to as the hostname. The computers themselves, however, utilize IP addresses which route to the site themselves, but for us as humans it is easier to remember a name to route to a given website for a given purpose. It is essentially an abstraction of the given IP address for that given site. This abstraction actually helps in another way that the fixed IP that is only tied to a given machine is not what is used to route to a site. The domain name can be tied to multiple machines and link between the domain and the IP can always be changed.

ICANN

ICANN or the Internet Corporation for Assigned Names and Numbers is an organization the is responsible for maintaining the procedures of several databases that are in accordance to the namespaces and numerical spaces of the internet, ensuring its stability. It was formed in 1998 and is a non-profit partnership of people from all over the world, dedicated to keeping the internet secure and stable. ICANN also maintaining the fact that no two IP addresses can be identical. They themselves do not run the system assigning IP addresses, but instead helped to coordinate how IP addresses are supplied to mitigate repetition or reuse.
